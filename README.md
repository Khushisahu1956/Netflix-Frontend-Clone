# Netflix Frontend Clone 🎬

A frontend clone of the Netflix website built using HTML, CSS, and JavaScript for learning and practice purposes.

## Features

* Responsive landing page UI
* Navigation bar
* Hero section
* Sign-in section
* FAQ section
* Footer design
* JavaScript-based interactive elements
* Mobile responsiveness improvements

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Status

Frontend completed
Responsiveness improved for multiple screen sizes
Built as a beginner project to strengthen frontend development skills

## Learning Outcome

This project helped improve understanding of responsive design, layout structuring, JavaScript interactivity, CSS styling, and frontend problem solving.
